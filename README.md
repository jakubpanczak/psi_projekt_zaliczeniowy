# psi_projekt_zaliczeniowy

‼️ **PROSIMY PRZECZYTAĆ NAJPIERW** ‼️

Nasz projekt zaliczeniowy zawiera w sobie interaktywny wykres z asocjacjami. 
Użytkownik ma możliwość wyboru interesującego go terminu i progu korelacji.
Z racji, że wykorzystaliśmy pakiet Shiny, niemożliwe było stworzenie **dynamicznego** raportu HTML.

W repozytorium zamieściliśmy raport HTML, który jest statyczny.
Nie zawiera on interaktywnego wykresu – dlatego prosimy o:
samodzielne uruchomienie kodu, aby zobaczyć końcowy rezultat. 😊
